The standard

xOPs

Procedures that hold a judgment instead of resolving it.

SOP executes  →  AOP executes autonomously  →  xOP checks whether the stance is still licensed, and holds.

Lyra gives you the version that doesn't perform. An xOP gives you the version that doesn't resolve.

The one rule
false_positive_on_warranted == 0

Never override a state that is still warranted. A real anger. A scoped refusal. A live caution. A genuine limit. The one rule that reads the same for a coach, a manager, an editor, and an agent.
